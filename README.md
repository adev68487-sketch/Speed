# Planetoid with light source

A Pen created on CodePen.

Original URL: [https://codepen.io/NiklasKnaack/pen/zQpzpb](https://codepen.io/NiklasKnaack/pen/zQpzpb).

You can see a planetoid consisting of a variable number of particles. The scene is illuminated by a light source. With the help of the menu you can load different textures and adjust the intensity of the bump map.
